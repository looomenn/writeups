## Structure

From now on, every ctf wu will be organised like this:

```shell
<year>
│
├─ fake-ctf/
│    │ 
│    ├─ assets/
│    ├─ README.MD
│    │ 
│    └ < category name >/
│          │
│          └ < chall name >/
│                │
│                ├─ assets
│                └─ README.md
```
